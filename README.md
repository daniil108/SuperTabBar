# SuperTabBar

Highly customizable SwiftUI TabBar, which allows you to create both simple tabbars and non-standard ones with a unique design.

## Installation

### SPM

Add https://github.com/daniil108/SuperTabBar using Package Dependecies.

### CocoaPods

```
pod 'SuperTabBar'
```
